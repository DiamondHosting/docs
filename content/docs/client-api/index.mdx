# 用戶端 API

所有對用戶端 API 的請求都應以 `/api/client` 開頭。
所有 API 請求皆以參照 [Pterodactyl API](https://pterodactyl-api-docs.netvpx.com/docs/api/client) 延伸。
本文檔僅涵蓋了我們修改後的面板中可用的自定義端點。

## 驗證
大多數端點需要有效的 API 金鑰或會話(Session)驗證。
詳見 [Pterodactyl API文檔](https://pterodactyl-api-docs.netvpx.com/docs/api/client#authentication)
