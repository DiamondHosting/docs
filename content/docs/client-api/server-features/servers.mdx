# 伺服器管理 (Server Management)
基礎路徑: `/api/client/servers/{server}`

本節中的所有端點都需要有效的伺服器識別碼 (UUID/Short UUID)。

## 核心控制 (Core Controls)
| 方法 | 端點 | 描述 |
| :--- | :--- | :--- |
| `GET` | `/` | 獲取伺服器詳細資訊 (狀態、限制等)。 |
| `GET` | `/websocket` | 獲取 WebSocket 憑證。 |
| `GET` | `/resources` | 獲取當前資源使用情況 (CPU, RAM, Disk)。 |
| `GET` | `/activity` | 獲取伺服器活動日誌。 |
| `GET` | `/traffic` | 獲取網路流量指標。 |
| `POST` | `/command` | 發送控制台指令。 |
| `POST` | `/power` | 執行特定的電源操作 (start, stop, restart, kill)。 |
