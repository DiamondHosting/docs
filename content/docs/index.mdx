# 選購指南

以下為 Minecraft 專屬伺服器的兩大方案比較，包含平價主機 (TW-B-1 ~ TW-B-9) 與極致主機 (TW-X-1 ~ TW-X-9)，請依照需求選擇最適合的方案與套餐。

## 方案比較

| 方案   | 處理器                                  | 記憶體          | 儲存           | 套餐編號               |
| ------ | --------------------------------------- | --------------- | -------------- | ---------------------- |
| 平價主機 | Intel Xeon E5-2660 v3 |  DDR4 | NVMe SSD | TW-B-1 ~ TW-B-9 |
| 極致主機 | Intel i9-14900KF | DDR5 5600 | Gen4 NVMe SSD | TW-X-1 ~ TW-X-9 |

---

## 選擇建議

1. **小型伺服器 (1-10 人)**：TW-B-1 ~ TW-B-3 或 TW-X-1 ~ TW-X-3  
2. **中型伺服器 (10-50 人)**：TW-B-4 ~ TW-B-6 或 TW-X-4 ~ TW-X-6  
3. **大型伺服器 (50 人以上)**：TW-B-7 ~ TW-B-9 或 TW-X-7 ~ TW-X-9
